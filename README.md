## Features
- ReactJS, NextJS, TypeSript
- Utilize Lazy Loading from NextJS (Image of NextJS support the Lazy Loading feature)

## Usage

It's an image gallery project that uses ReactJS/TypeScript/NextJS, so just do:
```shell
yarn install
yarn dev
```
